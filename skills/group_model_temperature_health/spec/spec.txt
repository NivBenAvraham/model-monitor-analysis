─────────────────────────────────────────────────────────────────
PURPOSE
─────────────────────────────────────────────────────────────────
Layer 2 of the temperature-based model monitoring pipeline.

Given the per-sensor outputs of sensor_group_segment (Layer 1),
decide whether the bee-frames model is working correctly for a
given (group_id, date).

Output: VALID / INVALID per (group_id, date).

─────────────────────────────────────────────────────────────────
INPUTS
─────────────────────────────────────────────────────────────────
- Layer 1 results: data/results/sensor_group_segment/
    Per-sensor features and PASS/WARNING/FAIL grades per (group, date).

- Ground truth labels: ground_truth/ground_truth_statuess_ca_2026.csv
    Human-labelled VALID / INVALID per (group_id, date).
    Used for calibration and evaluation.

- Train/test split: data/samples/split_manifest.csv
    75/25 stratified split of (group_id, date) pairs.
    Valid + invalid only (needs_recalibration excluded).

─────────────────────────────────────────────────────────────────
DATA UNIT
─────────────────────────────────────────────────────────────────
Each data point is one (group_id, date) pair — ~565 labelled pairs total.
  Train: 423 pairs (valid=269, invalid=154)
  Test:  142 pairs (valid=90, invalid=52)
